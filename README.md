# Playerok Telegram Bot

Telegram бот для работы с API торговой площадки Playerok.

## Возможности

- Получение списка всех товаров с вашего аккаунта Playerok
- Просмотр информации о товарах (название, цена, игра, статус)
- Поиск товаров по названию
- Пагинация списка товаров (15 товаров на страницу)
- Просмотр чатов с покупателями
- Отправка сообщений в чаты через бота
- Уведомления в реальном времени о:
  - Новых сообщениях от покупателей
  - Покупках товаров
  - Подтверждении сделок
  - Новых отзывах
  - Проблемах в сделках
- Логирование всех действий

## Установка

1. Установите Python 3.11 или выше
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка

1. Откройте файл `config.py`
2. Заполните необходимые данные:
   - `TELEGRAM_BOT_TOKEN` - токен вашего бота (получить у @BotFather)
   - `PLAYEROK_TOKEN` - токен вашего аккаунта Playerok
   - `ADMIN_IDS` - список Telegram ID администраторов (для получения уведомлений)

### Как получить Playerok токен:

1. Откройте сайт playerok.com в браузере
2. Войдите в свой аккаунт
3. Откройте DevTools (F12)
4. Перейдите на вкладку Application/Storage → Cookies
5. Найдите cookie с именем `token` и скопируйте его значение

### Как узнать свой Telegram ID:

Напишите боту @userinfobot в Telegram

## Запуск

```bash
python bot.py
```

## Команды бота

- `/start` - Начать работу с ботом
- `/products` - Получить список всех товаров
- `/chats` - Открыть список чатов
- `/search` - Поиск товара по названию
- `/help` - Показать справку

## Кнопки

- 📦 Мои товары - список активных товаров с пагинацией
- 💬 Чаты - список чатов с покупателями
- 🔍 Поиск товара - поиск по названию
- ❓ Помощь - справка

## Уведомления

Бот автоматически отправляет уведомления администраторам (указанным в ADMIN_IDS) о:
- Новых сообщениях от покупателей (с кнопкой "Открыть чат")
- Новых покупках товаров
- Подтверждении сделок
- Новых отзывах
- Проблемах в сделках

## Работа с чатами

1. Нажмите "💬 Чаты" для просмотра списка чатов
2. Выберите чат для просмотра последних сообщений
3. Нажмите "✍️ Ответить" и напишите сообщение в Telegram
4. Сообщение будет отправлено покупателю на Playerok
